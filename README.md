# my_library
## Kitap Takip Uygulaması

`My Library`, kitaplarınızı, yazarlarınızı ve okuma süreçlerinizi verimli bir şekilde yönetmenizi sağlayan, Python tabanlı bir konsol uygulamasıdır. Veritabanı olarak **SQLite3** kullanır ve kullanıcıların okuma alışkanlıklarını takip etmelerini kolaylaştırır.

## Özellikler

- **Kitap Yönetimi:** Kütüphanenize yeni kitaplar ekleyin, silin ve listeleyin.
- **Puanlama Sistemi:** Okuduğunuz kitapları 0-100 arası puanlayın.
- **Yazar Analizi:** Yazarların, okuduğunuz kitaplar üzerinden genel beğeni seviyelerini otomatik hesaplayın.
- **Veri Odaklı:** Tüm veriler yerel bir SQLite veritabanında güvenle saklanır.

## Kullanılan Teknolojiler

- **Dil:** Python 3.x
- **Veritabanı:** SQLite3
- **Arayüz:** Konsol (CLI)
